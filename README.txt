== Description ==

This is a lightweight plugin to create a customized stamp duty form for
calculating the stamp duty in England and Northern Ireland. 

You have the option to calculate for new properties, first time buyers
and second home owners


Add shortcode [njl-stamp-duty] on a page. Use the stamp duty menu option 

to
set the field text names

== Installation ==
From your WordPress admin 
    Navigate to the ‘Add New’ in the plugins dashboard
    Search for ‘\"njl stamp duty\"
    Click ‘Install Now’ button
    Activate the plugin in the Plugin dashboard
